# toolbox plugin

Manage and discover ToolBox plugins.

Aliases: `toolbox plugins`

## Subcommands

- [toolbox plugin create](Command-plugin-create) — Create a new plugin scaffold.
- [toolbox plugin install](Command-plugin-install) — Install a community plugin (pip wrapper).
- [toolbox plugin list](Command-plugin-list) — List all installed plugins.
- [toolbox plugin search](Command-plugin-search) — Search for community plugins.
