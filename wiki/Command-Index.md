## Command Groups

- [toolbox ai](Command-ai) — AI model management and edge computing tools.
- [toolbox ambient](Command-ambient) — Ambient Intelligence and Natural Interface tools.
- [toolbox archive](Command-archive) — Archive and compression tools (zip, tar, etc.).
- [toolbox audio](Command-audio) — Audio processing tools (requires FFmpeg).
- [toolbox config](Command-config) — Manage ToolBox configuration.
- [toolbox data](Command-data) — Data format conversion and inspection.
- [toolbox desktop](Command-desktop) — Desktop integration tools.
- [toolbox doc](Command-doc) — Document conversion and inspection tools.
- [toolbox file](Command-file) — File management utilities.
- [toolbox image](Command-image) — Image processing tools.
- [toolbox network](Command-network) — Network and web utilities.
- [toolbox pdf](Command-pdf) — PDF management tools.
- [toolbox plugin](Command-plugin) — Manage and discover ToolBox plugins.
- [toolbox security](Command-security) — Security and privacy tools.
- [toolbox util](Command-util) — General utility tools.
- [toolbox video](Command-video) — Video processing tools (requires FFmpeg).
- [toolbox workflow](Command-workflow) — Manage and run automated workflows.

## Top-Level Commands

- [toolbox check](Command-check) — Thoroughly check system readiness and environment.
- [toolbox status](Command-status) — Check the status of engines and plugins.
- [toolbox update](Command-update) — Self-update ToolBox from the official repository.
