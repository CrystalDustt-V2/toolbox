# toolbox desktop daemon

Start the ToolBox desktop daemon for background tasks.

Source: [src/toolbox/plugins/desktop/__init__.py:L36](https://github.com/CrystalDustt-V2/toolbox/blob/master/src/toolbox/plugins/desktop/__init__.py#L36)

## Synopsis

```bash
toolbox desktop daemon [--tray] [--hotkeys]
```

## Examples

```bash
toolbox desktop daemon --tray --hotkeys
```

Usage: toolbox desktop daemon [OPTIONS]
