# toolbox network

Network and web utilities.

## Subcommands

- [toolbox network cert-check](Command-network-cert-check) — Check SSL certificate information.
- [toolbox network download](Command-network-download) — Download a file from a URL.
- [toolbox network fleet-api](Command-network-fleet-api) — Start the Local API Gateway for remote fleet management.
- [toolbox network fleet-dispatch](Command-network-fleet-dispatch) — Dispatch a ToolBox command to a remote worker node.
- [toolbox network fleet-parallel](Command-network-fleet-parallel) — Hyper-Scale Parallelism: Split a batch of files across all fleet nodes.
- [toolbox network fleet-status](Command-network-fleet-status) — Scan for active ToolBox nodes on the local network.
- [toolbox network fleet-worker](Command-network-fleet-worker) — Start a distributed worker to accept remote tasks.
- [toolbox network info](Command-network-info) — Get local and public network information.
- [toolbox network mesh-sync](Command-network-mesh-sync) — Global Mesh Sync: DHT-based synchronization for vaults across locations.
- [toolbox network mycelium](Command-network-mycelium) — Biological Logic: Mycelial Fleet management for adaptive resource sharing.
- [toolbox network p2p-receive](Command-network-p2p-receive) — Receive a file from another ToolBox instance.
- [toolbox network p2p-send](Command-network-p2p-send) — Send a file to another ToolBox instance on the network.
- [toolbox network ping](Command-network-ping) — Basic host availability check.
- [toolbox network scan](Command-network-scan) — Simple port scanner.
- [toolbox network serve](Command-network-serve) — Start a simple HTTP server for a directory.
- [toolbox network whois](Command-network-whois) — Get WHOIS information for a domain (via RDAP).
