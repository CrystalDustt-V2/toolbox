# toolbox pdf

PDF management tools.

## Subcommands

- [toolbox pdf extract-text](Command-pdf-extract-text) — Extract text content from a PDF. Supports local or URL.
- [toolbox pdf merge](Command-pdf-merge) — Merge multiple PDF files into one. Supports local or URL.
- [toolbox pdf metadata](Command-pdf-metadata) — View or strip PDF metadata. Supports local or URL.
- [toolbox pdf ocr](Command-pdf-ocr) — Perform OCR on a PDF. Supports local or URL.
- [toolbox pdf rotate](Command-pdf-rotate) — Rotate all pages in a PDF. Supports local or URL.
- [toolbox pdf sanitize](Command-pdf-sanitize) — Remove metadata and hidden information from a PDF.
- [toolbox pdf split](Command-pdf-split) — Split a PDF into individual pages. Supports local or URL.
