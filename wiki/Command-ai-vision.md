# toolbox ai vision

Analyze an image using a local Vision LLM (LLaVA).

Source: [src/toolbox/plugins/ai/__init__.py:L90](https://github.com/CrystalDustt-V2/toolbox/blob/master/src/toolbox/plugins/ai/__init__.py#L90)

## Synopsis

```bash
toolbox ai vision IMAGE_PATH PROMPT [--model TEXT]
```

## Examples

```bash
toolbox ai vision <image_path> <prompt>
toolbox ai vision --model <model> <image_path> <prompt>
```

## Help

```text
Usage: toolbox ai vision [OPTIONS] IMAGE_PATH PROMPT

  Analyze an image using a local Vision LLM (LLaVA).

Options:
  --model TEXT  Vision model to use
  --help        Show this message and exit.
```
