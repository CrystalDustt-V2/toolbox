# toolbox config

Manage ToolBox configuration.

## Subcommands

- [toolbox config list](Command-config-list) — List all current settings.
- [toolbox config set](Command-config-set) — Update a setting (key value). Use dots for nested keys (e.g. engine_paths.ffmpeg).
