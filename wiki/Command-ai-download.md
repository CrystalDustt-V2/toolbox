# toolbox ai download

Pre-cache AI models for offline use.

Source: [src/toolbox/plugins/ai/__init__.py:L183](https://github.com/CrystalDustt-V2/toolbox/blob/master/src/toolbox/plugins/ai/__init__.py#L183)

## Synopsis

```bash
toolbox ai download [--model TEXT] [--all]
```

## Examples

```bash
toolbox ai download --model <model> --all
```

## Help

```text
Usage: toolbox ai download [OPTIONS]

  Pre-cache AI models for offline use.

Options:
  --model TEXT  Specific model to download (e.g.,
                upscale-x2, upscale-x4)
  --all         Download all available models
  --help        Show this message and exit.
```
