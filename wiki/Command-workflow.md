# toolbox workflow

Manage and run automated workflows.

## Subcommands

- [toolbox workflow init](Command-workflow-init) — Interactively create a new workflow file.
- [toolbox workflow run](Command-workflow-run) — Run a sequence of commands from a workflow YAML file.
- [toolbox workflow schedule](Command-workflow-schedule) — Schedule a workflow to run periodically.
- [toolbox workflow watch](Command-workflow-watch) — Watch a directory for new files and auto-trigger a workflow.
