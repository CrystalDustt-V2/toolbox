# toolbox network serve

Start a simple HTTP server for a directory.

Source: [src/toolbox/plugins/network/__init__.py:L61](https://github.com/CrystalDustt-V2/toolbox/blob/master/src/toolbox/plugins/network/__init__.py#L61)

## Synopsis

```bash
toolbox network serve [-p INTEGER] [-d PATH]
```

## Examples

```bash
toolbox network serve -p <port> -d <directory>
```
