# toolbox data

Data format conversion and inspection.

## Subcommands

- [toolbox data convert](Command-data-convert) — Convert between JSON, CSV, and YAML. Supports local or URL.
- [toolbox data inspect](Command-data-inspect) — Inspect data structure and summary. Supports local or URL.
- [toolbox data sql-export](Command-data-sql-export) — Export JSON or CSV data to a SQLite database.
