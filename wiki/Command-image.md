# toolbox image

Image processing tools.

## Subcommands

- [toolbox image convert](Command-image-convert) — Convert image format (e.g., photo.jpg photo.png). Supports local or URL.
- [toolbox image crop](Command-image-crop) — Crop an image (left, top, right, bottom). Supports local or URL.
- [toolbox image exif-strip](Command-image-exif-strip) — Remove EXIF metadata from an image. Supports local or URL.
- [toolbox image metadata](Command-image-metadata) — View image EXIF metadata. Supports local or URL.
- [toolbox image ocr](Command-image-ocr) — Perform OCR on an image. Supports local or URL.
- [toolbox image remove-bg](Command-image-remove-bg) — Remove image background using rembg (AI-powered).
- [toolbox image resize](Command-image-resize) — Resize an image. Supports local or URL.
- [toolbox image to-sticker](Command-image-to-sticker) — Convert an image or GIF to a sticker. Supports local or URL.
- [toolbox image upscale](Command-image-upscale) — Upscale image using AI (ESRGAN). Supports local or URL.
