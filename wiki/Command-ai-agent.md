# toolbox ai agent

Run an autonomous AI agent to achieve a goal using ToolBox commands.

Source: [src/toolbox/plugins/ai/__init__.py:L211](https://github.com/CrystalDustt-V2/toolbox/blob/master/src/toolbox/plugins/ai/__init__.py#L211)

## Synopsis

```bash
toolbox ai agent GOAL [--model TEXT]
```

## Examples

```bash
toolbox ai agent <goal>
toolbox ai agent --model <model> <goal>
```
