# toolbox doc

Document conversion and inspection tools.

## Subcommands

- [toolbox doc convert](Command-doc-convert) — Convert document format using LibreOffice. Supports local or URL.
- [toolbox doc inspect](Command-doc-inspect) — Get basic information about a document. Supports local or URL.
