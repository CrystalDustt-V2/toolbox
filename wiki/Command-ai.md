# toolbox ai

AI model management and edge computing tools.

## Subcommands

- [toolbox ai agent](Command-ai-agent) — Run an autonomous AI agent to achieve a goal using ToolBox commands.
- [toolbox ai bci](Command-ai-bci) — Brain-Computer Interface (BCI) Foundation: Simulated thought-to-command execution.
- [toolbox ai chat](Command-ai-chat) — Chat with a local LLM (offline, supports RAG).
- [toolbox ai download](Command-ai-download) — Pre-cache AI models for offline use.
- [toolbox ai image-gen](Command-ai-image-gen) — Generate an image from text using local Stable Diffusion (ONNX).
- [toolbox ai index](Command-ai-index) — Index local documents for RAG-enabled chat.
- [toolbox ai singularity](Command-ai-singularity) — ToolBox Singularity: Unified Intelligence Core. Orchestrates all Phase 1-9 systems.
- [toolbox ai vision](Command-ai-vision) — Analyze an image using a local Vision LLM (LLaVA).
