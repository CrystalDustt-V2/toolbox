# toolbox desktop

Desktop integration tools.

## Subcommands

- [toolbox desktop ar-overlay](Command-desktop-ar-overlay) — Holographic/AR CLI API Hooks: Simulate AR output overlay for ToolBox data.
- [toolbox desktop daemon](Command-desktop-daemon) — Start the ToolBox desktop daemon for background tasks.
- [toolbox desktop dashboard](Command-desktop-dashboard) — Start the local ToolBox dashboard.
- [toolbox desktop install-context-menu](Command-desktop-install-context-menu) — [Windows] Add ToolBox to the file context menu (Right-Click).
- [toolbox desktop notify](Command-desktop-notify) — Send a native desktop notification.
- [toolbox desktop register-file-type](Command-desktop-register-file-type) — [Windows] Associate a file extension with ToolBox.
- [toolbox desktop uninstall-context-menu](Command-desktop-uninstall-context-menu) — [Windows] Remove ToolBox from the file context menu.
