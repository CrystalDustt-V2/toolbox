# toolbox archive

Archive and compression tools (zip, tar, etc.).

## Subcommands

- [toolbox archive compress](Command-archive-compress) — Compress a file or directory.
- [toolbox archive extract](Command-archive-extract) — Extract an archive file. Supports local or URL.

## Help
