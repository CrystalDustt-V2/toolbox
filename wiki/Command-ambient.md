# toolbox ambient

Ambient Intelligence and Natural Interface tools.

## Subcommands

- [toolbox ambient fleet-health](Command-ambient-fleet-health) — Monitor fleet health and enable self-healing re-routing.
- [toolbox ambient predict](Command-ambient-predict) — Analyze usage patterns and suggest automated workflows.
- [toolbox ambient shadow](Command-ambient-shadow) — Predictive Pre-Computation: Shadow-process files before user requests them.
- [toolbox ambient voice](Command-ambient-voice) — Listen for a voice command and execute it via AI Agent.
