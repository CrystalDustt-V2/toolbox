# toolbox util

General utility tools.

## Subcommands

- [toolbox util base64](Command-util-base64) — Base64 encoding/decoding.
- [toolbox util case](Command-util-case) — Change text case.
- [toolbox util count](Command-util-count) — Count words, characters, and lines.
- [toolbox util evolve](Command-util-evolve) — Genetic Algorithm Workflow Optimizer: Benchmark and evolve workflows for efficiency.
- [toolbox util password](Command-util-password) — Generate a secure random password.
- [toolbox util qr](Command-util-qr) — Generate a QR code from text.
- [toolbox util regex](Command-util-regex) — Extract text using regex pattern. Supports local or URL.
- [toolbox util replace](Command-util-replace) — Find and replace text in a file. Supports local or URL.
- [toolbox util sort](Command-util-sort) — Sort lines in a text file. Supports local or URL.
- [toolbox util url](Command-util-url) — URL encoding/decoding.
- [toolbox util workflow](Command-util-workflow) — Run a YAML workflow with dynamic logic (if/else, variables).
